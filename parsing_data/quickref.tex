{\ex \lstinline|    str_var.split()|}

Splitting a string using a delimiter. Ex: ";" :\\
{\ex \lstinline|    str_var.split(";")|}

{\ex \lstinline|    str_var.strip()|}

{\ex \lstinline|    int(var_name)|}
